# n1nail-ca

Modern Next.js storefront for N1 Nail built by Zenith Strategic Solutions.

## Repo & Deployment

- GitHub: [zssca/n1nail-ca](https://github.com/zssca/n1nail-ca)
- Local directory: `n1nail-ca`
- Vercel project: `n1nail-ca`

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start developing. Build and test with:

```bash
npm run build
npm run start
```

The project uses Next.js 15, Tailwind CSS, and shadcn/ui. Configure environment secrets in `.env.local` before deploying to the matching Vercel project.
